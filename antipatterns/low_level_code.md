# Low level code

Код не разделен на инфраструктуру и бизнес код. Например, вынесение UI-кита как раз и есть выделение инфраструктуры, с которой писать код легче и быстрее. 

<img width="1039" height="896" alt="image" src="https://github.com/user-attachments/assets/f4dc8b3a-f7c2-4820-8152-884fb3971b45" />


Техническую сложность легко переиспользовать.

**Проблемы low level code**

1. Много лишнего кода
2. Изменение требований требуют больших изменений в коде
3. Сложность размазана по проекту. И бизнес сложность сложная, и техническая сложность тоже, но если это не разделено, то это сложность в квадрате.

**Как исправить?**

1. Использование библиотек.
2. Написание своей инфраструктуры.
